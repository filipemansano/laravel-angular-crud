## Angular Application

this application makes use of the following features of the framework

- Modules
- Components
- Service
- Guard
- Routes
- Interceptors

NgRX
- action
- reduce
- store
- effect
- entity
- data

this project use Angular Material / PrimeNG components and Tailwind utilities
